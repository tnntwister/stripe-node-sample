# stripe-node-sample

## how to use 

1. create a .env file, with two variables 

```
STRIPE_TEST_PK="pk_test_XXXXX"
STRIPE_TEST_SK="sk_test_XXXXXXXXXX"
```

2. install npm dependancies 

```
npm install
```

3. launch project 

```
nodemon index.js
```

4. open your bro(wser)

go to [http://localhost:3000/](http://localhost:3000/)
